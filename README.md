# 🎓 Magshimim – 9-10th Grade Homework & Projects (2019)

This repository contains homework assignments, exercises, and projects completed during my participation in the **Magshimim (מגשימים)** program — a cybersecurity and computer science program for high school students.

---

## 📚 Topics Covered

Throughout this program, I worked on:

* 🧠 **C Programming**

  * Variables, conditions, loops
  * Functions and memory basics
  * Problem-solving exercises
  * Bainry search and much more.

* 🐍 **Python**

  * Basic scripting
  * Working with data structures
  * Writing small automation programs

* 🌐 **Networking Basics**

  * Understanding network models
  * Protocols and communication
  * Introductory cybersecurity concepts
  * CTF and Pentesting
